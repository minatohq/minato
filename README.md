# Minato

## Development

### Ports

Development ports are assigned by application type and remain stable once assigned.

- Frontend applications use ports `3000-3099`.
- Backend services use ports `4000-4099`.

| Application | Port |
| --- | ---: |
| Website | 3000 |
| Dashboard | 3001 |
| Widget | 3002 |
| API | 4000 |
