# AGENTS.md

This file provides guidance to AI coding agents when working with code for this app

## Development

### Tech stack

- TanStack Start as the fullstack framework
- React as the frontend library
- TypeScript as primary language
- Shadcn UI and Base UI for base components
- Tailwind CSS for styling

#### React

- This project uses React 19 with React Compiler enabled
- Do not use useMemo and useCallback hooks, or memo function
